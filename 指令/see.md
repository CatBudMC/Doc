# /see
控制[動態地圖](/物品/動態地圖.md)的顯示中心。

## /see `行為`
| 參數 | 中心點 | 偏移 |
| :---: | :---: | :---: |
| `up` / `u` | 向上 | `-Z` |
| `down` / `d` | 向下 | `+Z` |
| `left` / `l` | 向左 | `-X` |
| `right` / `r` | 向右 | `+X` |

## /see `座標X` `座標Y` `座標Z`
座標可以使用 `~` 表示與自己的相對位置、`^` 表示基於當前的相對位置，或者直接輸入精確位置。`~-10 ~ ~` 表示距離自身 X軸 `-10` 方塊、`^ 20 ^` 使用當前位置但高度為 Y軸 `20` 方塊。
