# 領地書
可以創建、編輯、更改領地的配置與身分組。

## 合成
- 鑽石 × `8` + 書 × `1` = **空白的領地書**  
- 領地書 × `1` + 空白的領地書 × `1` = **領地書** + 空白的領地書

*（獲得材料時會動解鎖合成表）*

## 配置
可以花費 `100` 來讓領地的可用量增加 `1`，可用量每多 `1` 時轉移費用也會貴 `20`。最大可創建 `20` 個身分組和加入 `1,000` 位玩家。

## 宣告
手持領地書時，畫面上會充滿需多節點，可能有：
| 圖示 | 含意 |
| :---: | --- |
| ❌ | 伺服器保護區 |
| ✔ | 沒有人宣告的區域 |
| 🔓 | 已被手中領地書宣告的區域 |
| 🔒 | 已被其他領地書宣告的區域 |
| 🔑 | 被共享的領地書清單 |
