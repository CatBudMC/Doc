# 方塊活動
[子區塊](https://zh.minecraft.wiki/w/区块格式#方块位置编码)每刻會觸發 `randomTickSpeed` 次檢查，檢查順序按照下列排序。

## 碎裂
方塊水平鄰近同時有熱源、冰源，會將方塊碎裂化。
| 熱源 | 冰源 |
| --- | --- |
| 岩漿、岩漿塊 | 冰、藍冰、霜冰、冰磚 |

| 原本 | 碎裂 |
| --- | --- |
| *任何可能的* | 裂紋化 |
| *任何可能的* | 鵝卵石化 |
| *任何可能的* | 粉末化 |
| 鵝卵石 | 礫石 |
| 青苔鵝卵石 | 礫石 |
| 礫石 | 沙 |
| 裂紋石磚 | 鵝卵石 |
| 青苔石磚 | 裂紋石磚 |
| 砂岩 | 沙 |
| 紅砂岩 | 紅沙 |
