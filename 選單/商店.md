# 商店
快速購買遊戲幣，共有 `5` 種金額可選：
- `2000 T`（新台幣 `200` 元）
- `5000 T`（新台幣 `500` 元）
- `10000 T`（新台幣 `1000` 元）
- `20000 T`（新台幣 `2000` 元）
- `40000 T`（新台幣 `4000` 元）

### 貨幣是甚麼？
伺服器標準貨幣有 `C` 幣與 `T` 幣：
- `C` 幣是免費貨幣、玩家間不可交易，可以從[任務](/選單/任務.md)獲得。
- `T` 幣是付費或幣、玩家間可交易，只能從此處購買。
  
`C` 幣與 `T` 幣的在購買時所需的費用是一樣的，因此文檔內只會意旨「遊戲幣」而不是限定其中一種。

### 該如何使用？
`C` 幣與 `T` 幣可以用在很多地方：

| --- | --- |
| `100` | [領地](/物品/領地書.md)可用量 + `1` |
| `50` | [輔助](/選單/輔助.md)增加視野距離 +`1` 小時 |
| `50` | [輔助](/選單/輔助.md)領地飛行 +`1` 小時 |
| `4` | [加載](/選單/加載.md)區塊 +`1` 小時 |
| `4,000` | 創建初級[貨幣](/選單/貨幣.md) |
| `12,000` | 創建普通[貨幣](/選單/貨幣.md) |
| `36,000` | 創建高級[貨幣](/選單/貨幣.md) |

除此之外還有不固定金額的：
- [領地](/物品/領地書.md)管理者轉移。
- [貨幣](/選單/貨幣.md)管理者轉移。
