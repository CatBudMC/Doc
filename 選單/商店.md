# 商店
快速購買遊商品，有 `5` 種貨幣金額、 `3` 種通行證可選：
- `2,000 C`（新台幣 `200` 元）
- `5,000 C`（新台幣 `500` 元）
- `10,000 C`（新台幣 `1,000` 元）
- `20,000 C`（新台幣 `2,000` 元）
- `40,000 C`（新台幣 `4,000` 元）
- `1,000 C` + 通行證 `1` 月（新台幣 `300` 元）
- `2,000 C` + 通行證 `2` 月（新台幣 `600` 元）
- `4,000 C` + 通行證 `4` 月（新台幣 `1,200` 元）

### 「C 幣」是甚麼？
伺服器標準貨幣有 `C` 幣與 `T` 幣：
| 類型 | 玩家交易 | 獲取方式 |
| :---: | :---: | --- |
| `C` | 可以 | 從此處購買 |
| `T` | 可以 | 完成[任務](/選單/任務.md) |
  
`C` 幣與 `T` 幣花費給伺服器所需的價格是一樣的，因此文檔內只會意旨「遊戲幣」而不是限定其中一種。

### 「通行證」是甚麼？
有效時間內獲得增益：
- 領地飛行。
- 擴展視野距離。
- 暱稱有效。
- 傳送費用便宜 `20% T`。
- 死亡時不會扣除 `2% T`。
