# 商店
快速購買遊商品，有 `5` 種貨幣金額、 `3` 種通行證可選：
- `2,000 C`（新台幣 `200` 元）
- `5,000 C`（新台幣 `500` 元）
- `10,000 C`（新台幣 `1,000` 元）
- `20,000 C`（新台幣 `2,000` 元）
- `40,000 C`（新台幣 `4,000` 元）
- `1,000 C` + 通行證 `1` 月（新台幣 `300` 元）
- `2,000 C` + 通行證 `2` 月（新台幣 `600` 元）
- `4,000 C` + 通行證 `4` 月（新台幣 `1,200` 元）

### 「C 幣」是甚麼？
伺服器標準貨幣有 `C` 幣與 `T` 幣：
| 類型 | 玩家交易 | 獲取方式 |
| :---: | :---: | --- |
| `C` | 可 | 從此處購買 |
| `T` | 不可 | 完成[任務](/選單/任務.md) |
  
`C` 幣與 `T` 幣花費給伺服器所需的價格是一樣的，因此文檔內只會意旨「遊戲幣」而不是限定其中一種。

### 「通行證」是甚麼？
每天重置 `6` 個任務，包含：
- `1` 個 `100 T` + `2 A` 簽到任務。
- `5` 個 `10 T` ~ `30 T` + `1 A`（`40%`機率）隨機任務。

### 該如何使用？
`C` 幣與 `T` 幣可以用在很多地方：
| 金額 | 用途 |
| ---: | --- |
| `5` | [加載](/選單/加載.md)區塊 +`1` 小時 |
| `50` | [輔助](/選單/輔助.md)增加視野距離 +`1` 小時 |
| `100` | [領地](/物品/領地書.md)可用量 + `1` |
| `100` | [輔助](/選單/輔助.md)領地飛行 +`1` 小時 |
| `500` | [暱稱](/選單/暱稱.md)立即更改 |
| `1,000` | [暱稱](/選單/暱稱.md)有效時效 + `30` 天 |
| `4,000` | 創建初級[貨幣](/選單/貨幣.md) |
| `12,000` | 創建普通[貨幣](/選單/貨幣.md) |
| `36,000` | 創建高級[貨幣](/選單/貨幣.md) |

除此之外還有不固定金額的：
- [領地](/物品/領地書.md)管理者轉移。
- [貨幣](/選單/貨幣.md)管理者轉移。
- [移動](/選單/移動.md)至指定座標。

### 在哪裡付款？
由於購買紀錄需要綁定遊戲帳號，因此只能從遊戲內的選單購買。
