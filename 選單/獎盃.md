# 獎盃
獲得獎盃後可以展示在自己頭上，自己可以選擇不看到自己的展示。

## 可得清單
| 名稱 | 條件 |
| --- | --- |
| **遊玩之冠** | ＂總遊玩時間＂是伺服器第一名 |
| **繁殖之冠** | ＂繁殖動物總數＂是伺服器第一名 |
| **狩獵之冠** | ＂造成傷害總量＂是伺服器第一名 |
| **旅行之冠** | ＂移動距離＂是伺服器第一名 |
| **捕魚之冠** | ＂捕獲魚量＂是伺服器第一名 |
| **屠龍之冠** | ＂擊殺終界龍數量＂是伺服器第一名 |
| **全成就** | 獲得全部成就 |
| **極限模式** | 沒有死亡過 |
| **極限模式全成就** | 在沒有死亡過的情況下獲得全部成就 |
| **愛貓者** | 沒有殺過貓 |
| **存在自信** | 死亡次數比殺過的終界龍還少 |
