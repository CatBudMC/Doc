# 移動
隨時移動到想去的座標，需要支付一定的 `T` 或 `A` 幣，但有幾個限制：
1. 必須至少曾經前往過該維度，才能在未來直切換到該維度。
2. 不允許從地獄移動到終界，或者從終界移動到地獄。

費用計算公式：

| 維度 | 費用 | 
| --- | --- |
| *沒有改變* | `兩個位置的距離 × 0.1` | 
| `主世界` → `地獄` | `40 ＋ 當前位置到 ( 目標位置 × 8 ) 的距離 × 0.1` | 
| `地獄` → `主世界` | `40 ＋ 當前位置到 ( 目標位置 ÷ 8 ) 的距離 × 0.1` | 
| `主世界` ↔ `終界` | `40 ＋ 當前位置到世界出生地的距離 × 0.1 ＋ 目標位置到世界出生地的距離 × 0.1` |

無論如何費用都不會少於 `10`，座標可以使用 `~` 來代表當前相對位置。  
每位玩家都有 `5000 A` 折抵卷可用。
