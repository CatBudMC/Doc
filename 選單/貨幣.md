# 貨幣
查看貨幣狀態或者創建自己的貨幣，只有當自己是管理者時才能看到流通量。

## 等級
| 名稱 | 字數 | 創建費用 | 轉移所有權費用 |
| --- | :---: | --- | --- |
| 頂級 | `1` | *無法創建* | *無法轉移* |
| 高級 | `2` | `36,000` | `18,000` |
| 普通 | `3` | `12,000` | `6,000` |
| 初級 | `4` | `4,000` | `2,000` |

## 上鎖
| 名稱 | 圖示 | 可以轉帳 | 管理者可以編輯 |
| --- | :---: | --- | --- |
| 無上鎖 | *無* | 可以 | 可以 |
| 上鎖中（不含管理者） | 🔐 | 僅在其中一方是管理者時可以 | 可以 |
| 上鎖中 | 🔒 | 不行 | 可以 |
| 禁止中 | ⛔ | 不行 | 不行 |

## 語言
以下為可用的語言清單。
| 名稱 | 必需的 |
| --- | --- |
| English（US） | 是 |
| 繁體中文（台灣） | 是 |
| 繁體中文（香港） | 否 |
| 简体中文（中国） | 否 |
| 日本語 | 否 |
